# 締め切り・タスク管理アプリ

`task_deadline_app_project_summary.md` の要件に沿って作成した、個人向けのシンプルなWebタスク管理アプリです。

## 機能

- タスクの追加・編集・削除
- 締め切り日時の設定
- 完了 / 未完了の切り替え
- 締め切り順ソート（未完了優先）
- フィルタ（今日 / 今週 / 期限切れ / 完了済み / 未完了）
- 通知タイミング設定（1日前 / 3時間前 / 1時間前 / なし）
- アプリ内トースト通知
- ブラウザ通知（許可した場合）
- ローカルストレージ保存
- 通知済み状態の保存（ページ再読み込み後の重複通知を抑制）

## 使い方

1. `index.html` をブラウザで開きます。
2. タイトル・締め切り・通知タイミングを入力して保存します。
3. 一覧から完了/編集/削除が行えます。
4. 「表示」ドロップダウンで条件別に確認できます。

## ファイル構成

- `index.html` : 画面レイアウト
- `styles.css` : スタイル
- `script.js` : タスク管理ロジックと通知機能
- `USER_GUIDE.md` : 実際の利用手順書
- `task_deadline_app_project_summary.md` : 企画概要
