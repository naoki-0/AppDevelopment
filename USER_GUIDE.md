# 締め切り・タスク管理アプリ 利用手順書

この手順書は、`index.html` で動作する「締め切り・タスク管理アプリ」を実際に使うためのガイドです。

---

## 1. 事前準備

### 必要なもの
- PC（Windows / macOS / Linux）
- モダンブラウザ（Google Chrome / Microsoft Edge / Firefox / Safari の最新版推奨）

### 重要な前提
- このアプリは **ローカル保存（localStorage）** を使います。
- 保存データは「同じブラウザ」「同じ端末」にのみ残ります。
- ブラウザの閲覧履歴データ削除やシークレットモードでは、保存が消える/保持されない場合があります。

---

## 2. 起動方法

1. プロジェクトフォルダ内の `index.html` を確認します。
2. `index.html` をブラウザで開きます。
   - 例: `index.html` をダブルクリック
   - 例: ブラウザにドラッグ&ドロップ
3. 画面上部に「締め切りタスク管理」と表示されれば起動完了です。

---

## 3. 画面構成

- **タスクを追加**: タイトル、締め切り、通知タイミングを入力
- **ダッシュボード**: 未完了、今日のタスク、期限切れ、完了済みの件数
- **タスク一覧**: 追加済みタスクの表示・操作
- **表示フィルタ**: すべて / 今日 / 今週 / 期限切れ / 完了済み / 未完了

---

## 4. 基本操作

### 4.1 タスクを追加する
1. 「タイトル」を入力
2. 「締め切り日時」を選択
3. 「通知タイミング」を選択（通知なし / 1日前 / 3時間前 / 1時間前）
4. 「保存」をクリック

> 保存後、タスクは締め切りが近い順で表示されます（未完了が先）。

### 4.2 タスクを編集する
1. タスク一覧で対象タスクの「編集」をクリック
2. フォームに既存値が入るので内容を変更
3. 「保存」をクリック
4. キャンセルする場合は「編集をキャンセル」をクリック

### 4.3 完了/未完了を切り替える
- タスク一覧で「完了にする」または「未完了に戻す」をクリック

### 4.4 タスクを削除する
1. タスク一覧で「削除」をクリック
2. 確認ダイアログで OK を選択

---

## 5. フィルタの使い方

「表示」ドロップダウンで一覧を絞り込めます。

- **すべて**: 全タスク
- **今日**: 今日が締め切りのタスク
- **今週**: 今日から7日以内のタスク
- **期限切れ**: 締め切りを過ぎた未完了タスク
- **完了済み**: 完了タスクのみ
- **未完了**: 未完了タスクのみ

---

## 6. 通知機能

### 6.1 アプリ内通知
- 締め切り接近時に、画面右上にトースト通知を表示します。

### 6.2 ブラウザ通知
- 初回に通知許可が求められる場合があります。
- 許可すると、ブラウザの通知としても表示されます。

### 6.3 通知の重複抑制
- 一度通知したタスクは、同じ通知タイミングで何度も出ないように制御しています。
- この状態もローカルに保存されます。

---

## 7. 実運用のおすすめ

- 毎朝「今日」フィルタで優先タスクを確認
- 期限が近い重要タスクは「1日前通知」に設定
- 完了したタスクはこまめに完了化して一覧を整理

---

## 8. データ管理・注意点

- 保存先はブラウザの localStorage です（サーバー保存ではありません）。
- 別ブラウザ・別端末には同期されません。
- ブラウザデータ削除でタスクが消える場合があります。

---

## 9. トラブルシューティング

### Q1. タスクが保存されない
- JavaScript が無効化されていないか確認
- シークレットモードでないか確認
- ブラウザ設定で localStorage が制限されていないか確認

### Q2. ブラウザ通知が出ない
- 通知許可が拒否されていないか確認
- OS側の通知設定でブラウザ通知がOFFになっていないか確認

### Q3. タスクが消えた
- ブラウザデータを削除した可能性があります
- localStorage保存のため、バックアップ機能は現時点ではありません
